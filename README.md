# dapa (धप्पा)
Digital Arrest Protection App

## create .env

```
cp example.env .env
```

## Build app container
```
docker compose up --build -d
```

## Access chatbot
```
http://localhost:8088/
```

## API Documents
```
http://localhost:8080/docs
```
